# Public Assets

## Required Files (Add These)

Before deploying, add these files to this directory:

### 1. Favicon
- `favicon.ico` (32x32 or 16x16)
- Generate at: https://favicon.io

### 2. App Icons (for PWA)
- `icon-192.png` (192x192)
- `icon-512.png` (512x512)
- `apple-touch-icon.png` (180x180)

### 3. Open Graph Image
- `og-image.png` (1200x630)
- Used for social media previews
- Should show your brand/logo

### 4. Logo (Optional)
- `logo.png` (recommended: 500x500)
- Used in structured data

## Quick Setup

### Option 1: Use Placeholder Images (Quick Start)
Use a tool like [Placeholder.com](https://placeholder.com/) temporarily:
- Create colored squares with text
- Replace with real assets later

### Option 2: Generate Professional Assets
Use these free tools:
1. **Favicon**: https://favicon.io/favicon-generator/
2. **App Icons**: https://www.pwabuilder.com/imageGenerator
3. **OG Image**: https://www.canva.com/ (use 1200x630 template)

## Current Status

All files referenced in the code are in `manifest.json` and `app/layout.tsx`. The app will work without these images, but they enhance:
- Browser tab appearance
- Social media sharing
- PWA installation
- Professional branding

## Note

The app will function perfectly without these assets. Add them when you're ready to polish the production version!
